Simple and easy for usage 2-WaySMS Messenger. This plugin allows you send bulk or single SMS messages and receive SMS answers to your SMS Number.

== Description ==

2-WaySMS allows send SMS messages with your own Sender ID. Sender ID (Company Name, Your Name, etc) - it can be numeric or alphanumeric (0-9, a-Z up to 11 characters). You just have to install plugin and type your account information from 2-WaySMS http://www.2-waysms.com . SMS plugin messenger with reliable SMS gateway connection. Sending to more than 220 countries. Local SMS numbers for messages receiving of more than 43 countries are available.

== Installation ==

1. Upload `2waysms` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 2-WaySMS Settings in admin panel
4. To activate SMS sending enter your Token and Number and click Save
5. For SMS receiving, copy generated link from Receiver script field
6. Log in to your 2-WaySMS account http://www.2-waysms.com/my / My numbers / Settings
7. Set [Forward all Inbound Messages to the following URL] and paste copied link
8. Plugin ready for SMS sending and receiving

== Frequently Asked Questions ==

= Where i can find Token and Number? =
First of all you need to open an account and purchase at least one SMS number http://www.2-waysms.com/receive.html . Token will be located on page of Settings of your number. Number is your purchased SMS Number.

= What number format for SMS Sending? = 

Accepted phone numbers format:
International format (with country code)
Without +, 00, 0 or -
Numbers only

= How i can send Bulk SMS? =

Separate numbers with ;
Example: 4444444;555555555;6666666;7777777

= How many SMS messages i can Send or Receive? =

In case of SMS sending and receiving there is no limits, you can send while you have a credits on your 2-WaySMS http://www.2-waysms.com balance.
